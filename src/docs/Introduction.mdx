# @kingsley/design-system

A portfolio-grade design system built on precision, not decoration.

---

## Philosophy

**Precision over decoration** — Every token, spacing value, and component variant exists for a reason. Nothing is added speculatively.

**Tokens are the system** — Colors, spacing, radii, shadows, motion, and states are all defined as design tokens via Style Dictionary. The components are an expression of those tokens, not the other way around.

**Accessible by default** — All interactive components meet WCAG 2.1 AA. Keyboard navigation, focus management, and screen reader support are treated as requirements, not enhancements.

## Installation

```bash
npm install @kingsley/design-system
```

---

## Basic usage

Wrap your app with `ThemeProvider`, then import any component:

```tsx
import { ThemeProvider } from '@kingsley/design-system'
import { Button } from '@kingsley/design-system'
import '@kingsley/design-system/styles'

export default function App() {
  return (
    <ThemeProvider defaultTheme="light">
      <Button variant="primary">Get started</Button>
    </ThemeProvider>
  )
}
```
