# Design Tokens

Tokens are the foundation of the system. Every visual property — color, spacing, radius, shadow, motion, state — is defined as a CSS custom property generated by Style Dictionary from the `tokens/base/` source files.

Use tokens via CSS variables (`var(--token-name)`) or Tailwind utilities where configured.

---

## State tokens

| Token | Value | Usage |
|---|---|---|
| `--state-hover-overlay-subtle` | `0.04` | Hover background opacity |
| `--state-hover-overlay-medium` | `0.08` | Stronger hover background opacity |
| `--state-focus-ring-color` | `#6366f1` | Focus ring color |
| `--state-focus-ring-width` | `2px` | Focus ring stroke width |
| `--state-focus-ring-offset` | `2px` | Focus ring offset from element |
| `--state-active-opacity` | `0.9` | Opacity during active/press |
| `--state-disabled-opacity` | `0.5` | Opacity for disabled elements |
